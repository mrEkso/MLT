### Installation

1.Copy the repository

```sh
git clone https://github.com/mrEkso/MLT.git
```

2.Deploying the project

```sh
python -m venv .venv
.venv\Scripts\activate
```

3.Install the required packages

```sh
pip install -r requirements.txt
```

4.Run the project

```sh  
python main.py
```
